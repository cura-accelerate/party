# 🐻 くまさんのパズル

くまのシルエットにピースをはめる、2人用ターン制パズルゲームです！

## 遊び方

1. プレイヤー1と2が交互にピースを選ぶ
2. ボード上のくまのシルエット内にピースを配置する
3. ピースが全部なくなったらゲーム終了
4. 多くのマスを埋めたプレイヤーが勝ち！

## 開発

```bash
npm install
npm start
```

## デプロイ (Vercel)

```bash
npm run build
```

Vercelに接続してGitHubリポジトリからデプロイできます。

## 技術スタック

- React 18
- CSS Modules
- Vercel (ホスティング)
